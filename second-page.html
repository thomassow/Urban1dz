Определения

  1. HTML
    язык разметки для создания структуры веб-страницы и представления контента.

      Используется для:

    • Структурирования
    • Отображения страницы
    • Отображения контента на странице
  2. CSS
    CSS — Cascading Style Sheets — это каскадные таблицы стилей. По сути — язык, который отвечает за описание внешнего вида HTML-документа.
    Подавляющее большинство современных веб-сайтов работают на основе связки HTML+CSS.

      Используется для Стилизации страницы:

    • Фона
    • Текста
    • Стиль шрифтов
    • Декоративных элементов
  3. JavaSkript
    это язык, который позволяет вам применять сложные вещи на web странице — каждый раз, когда на web странице происходит что-то большее, чем просто её статичное отображение

      Позволяет создать:

    • Динамически обновляемый контент
    • Управлять мультимедиа
    • Анимировать изображения
    • Передавать информацию между пользователем и сервером, и многое другое.
